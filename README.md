# Dotfiles

My zsh/tmux/neovim configurations.

## Setup

Clone repo

```sh
git clone https://github.com/drewxs/dotfiles ~/.dotfiles
```

Install

For Debian

```sh
source  ~/.dotfiles/scripts/debian/install.sh
```

For Arch

```sh
source ~/.dotfiles/scripts/arch/install.sh
```
