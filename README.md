# Dotfiles

My alacritty/neovim/tmux/zsh configurations.

## Installation

Clone repo

```bash
git clone https://github.com/drewxs/dotfiles $HOME/.dotfiles
```

Install zsh/omz

```bash
source $HOME/.dotfiles/scripts/install_zsh.sh
```

Run from zsh

```bash
install_dotfiles
```

Run `upd -h` for update options.
