# Dotfiles

My dev configurations.

- wezterm
- zsh
- tmux
- neovim

## Installation

Clone repo

```sh
git clone https://github.com/drewxs/dotfiles ~/.dotfiles
```

Pre-installation

```sh
~/.dotfiles/scripts/preinstall.sh
```

Install

```sh
# Arch
~/.dotfiles/scripts/install_arch.sh

# Debian
~/.dotfiles/scripts/install_debian.sh
```

Run `upd -h` for update options.
