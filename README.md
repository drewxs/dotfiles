# Dotfiles

My dev configurations.

- wezterm
- zsh
- tmux
- neovim

## Installation

Clone repo

```sh
git clone https://github.com/drewxs/dotfiles ~/.dotfiles
```

Install

```sh
~/.dotfiles/scripts/install.sh
```

Run `upd -h` for update options.
