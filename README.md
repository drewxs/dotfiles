# Dotfiles

My zsh/tmux/neovim configurations.

## Setup

Clone repo

```sh
git clone https://github.com/drewxs/dotfiles ~/.dotfiles
```

Install

```sh
source  ~/.dotfiles/scripts/install.sh
```
